## Nota sobre "parque para perros" (9900 vol)

Keyword con mayor volumen detectada pero **NO incluida** en el plan por intento navegacional. Los usuarios buscan "parque para perros" para encontrar parques cerca de ellos (Google Maps). Las keyword ideas confirman: "parque para perros cerca de mi" 1900, "parque canino cerca de mi" 1600.

Un articulo informativo ("Guia de parques para perros en Espana") tendria dificultad extrema para posicionar contra resultados de Maps y directorios locales. El ROI no justifica el esfuerzo.

---

## Nota sobre estacionalidad

**Si, la publicacion estacional es buena estrategia.** Google favorece contenido indexado 4-6 semanas antes del pico de busquedas. El v7 mantiene este principio del v6:

- Contenido **evergreen** se publica cuando el volumen lo justifica (alta prioridad primero)
- Contenido **estacional** se calendariza por fecha de pico, no por volumen
- Algunos articulos evergreen tienen timing dual: "banar perro" es evergreen pero sube en verano

Lo que **no** tiene sentido es forzar articulos de 0 vol solo por timing estacional (como hacia el v6 con "cuidados mascotas verano" = 0 vol). Si no hay busquedas, no hay pico estacional que aprovechar.
