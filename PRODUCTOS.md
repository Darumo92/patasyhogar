# Lista de productos para actualizar

Para cada producto haz lo siguiente en Amazon.es:
1. Busca el producto por nombre
2. Abre la página del producto específico
3. Copia la URL completa de la barra del navegador (contiene el ASIN: `/dp/XXXXXXXXX`)
4. Haz click derecho en la imagen principal → "Copiar dirección de imagen"
5. Rellena los campos URL e Imagen debajo de cada producto

**Leyenda:**
- ✅ URL = ya tiene enlace directo con ASIN (`/dp/`)
- ✅ Imagen = ya tiene campo `imagen` relleno
- (campo vacío) = pendiente de rellenar
